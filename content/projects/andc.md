[www.andc.tv](https://www.andc.tv) (&C) is een website voor het gelijknamige tijdschrift van Chantal Janzen.  
Deze website is gebouwd in [SvelteKit](https://kit.svelte.dev) en ik ben erg tevreden over hoe het project is opgezet.

We hebben veel geleerd over [Contentful](https://www.contentful.com/), [Shopify](https://shopify.dev/) en [Buckaroo](https://www.buckaroo.nl/), deze worden gebruikt voor het beheren van content, producten & abonnementen.  
We liepen tegen veel beperkingen aan, deze systemen zijn niet per se slecht, maar ik had hogere verwachtingen. Authenticatie loopt via [AD B2C](https://docs.microsoft.com/nl-nl/azure/active-directory-b2c/); dit systeem is een drama en ik raad deze af voor vervolgprojecten.

Voor dit project werd [Scrum](https://www.scrum.org/) gebruikt, ik vond dit een fijne werkwijze.
