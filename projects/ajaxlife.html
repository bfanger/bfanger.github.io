Ajaxlife API

Mijn collega's hebben de nieuwe site van ajaxlife.nl gemaakt in wordpress. Ajaxlife heeft ook een iOS en Android app die gebruik maken van een api op de site.

Ik heb interface van de api van de vorige website nagemaakt, maar nu haalt deze zijn informatie uit de wordpress database. Ook heb ik dit project omgezet naar de vue-cli configuratie die ik eerder voor wordpress had geschreven. Dit was nodig omdat de op sage gebaseerde configuratie veel problemen gaf.
