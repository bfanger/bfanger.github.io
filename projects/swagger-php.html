Swagger-PHP

Swagger-PHP is een library waarmee je door php doc-comments (annotations) in de code te plaatsen een documentatie voor je REST API kunt genereren. De documentatie wordt weergegeven in swagger-ui waaruit je ook direct je API call kunt testen.

Wat begon met een paar patches is uitgegroeit tot een core-developer rol voor de open source library Swagger-PHP.
