&C (andc)

www.andc.tv ( &C ) een website voor het gelijknamige tijdschrift van Chantal Jansen.
Deze website is gebouwt in SvelteKit en ben erg tevreden over hoe het project is opgezet.

We hebben een hoop geleerd over Contentful, Shopify en Buckaroo, deze worden gebruikt voor het beheren van content, producten & abonnementen.
We liepen tegen een hoop beperkingen aan, deze systemen zijn niet perse slecht, maar ik had hogere verwachtingen. Authenticatie loopt via AD B2C, dit is een drama systeem en raad deze af voor vervolgprojecten.

Voor dit project werd Scrum gebruikt, ik vond dit een fijne werkwijze.
