Multiplayer Dice Game

Multiplayer Dice Game is een online multiplayer dobbel spel.

Voor het realtime multiplayer aspect van de game wordt socket.io gebruikt, om de livereload workflow te kunnen behouden worden de gespeelde spellen in redis opgeslagen en worden de zetten via pub/sub verstuurd naar de andere spelers.

Bijzonder is dat ipv events wordt de gehele gamestate verstuurd. de client vergelijkt deze state dan met eventuele vorige state en berekend welke events er waren.
