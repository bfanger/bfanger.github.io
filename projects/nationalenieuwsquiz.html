Radio 1 - Nationale Nieuws Quiz

Voor dit project hebben we een facebook pagina gemaakt waar je zelf De Nationale Nieuwsquiz kunt spelen. De Nationale Nieuwsquiz is een onderdeel van "De Ochtend" een programma op NPO Radio 1

Dit is mijn tweede project met het vue.js framework en dat bevalt prima, het is een leuke mix tussen Angular en React in.
